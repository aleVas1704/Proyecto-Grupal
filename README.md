###Que es GIT?

Git es un sistema de control de versiones, crea puntos de guardado para
nuestros archivos teniendo un control de versiones y permitiendonos volver
a versiones anteriores, tambien es una herramienta grupal y por eso es un 
sistema de control distribuido

###Como nacio GIT?

surgio de la necesidad buscar una mejor forma de guardar el repositorio 
de actualizaciones de linus torvalds, en un principio el uso BITKEEPER el 
cual tuvo un acuerdo con el con respecto a que ni el ni sus colaboradores 
usaran otro ademas de este, pasado un tiempo se rompio ese acuerdo y se le 
revocaron los derechos a linus, siendo asi este enojado se quedo como 2 o 3
semanas en su casa y con ayuda de diversas herramientas creo GIT.
